## [v0.1.0] - 2026-01-25

### Highlights

- Initial release of agent-team-content: multi-agent system for content generation
- 6 specialized agents for transforming conversations into multiple content formats
- Integration with Claude API for AI-powered content generation

### Added

- content CLI for content generation: generate, list-agents, version commands
- blog agent for Medium/Substack articles
- devto agent for technical articles on dev.to
- linkedin agent for professional LinkedIn posts
- twitter agent for Twitter/X threads
- marp agent for Marp Markdown presentations with speaker notes
- revealjs agent for Reveal.js presentations
- internal/agent package with orchestrator for parallel agent execution
- internal/conversation package for parsing JSON and Markdown conversations
- internal/llm package for Claude API integration via anthropic-sdk-go
- Agent specifications in specs/agents/ with YAML frontmatter
- Team workflow specification for parallel content generation
- AssistantKit integration for generating Claude Code and Kiro agents

### Dependencies

- github.com/agentplexus/assistantkit v0.7.0 for agent generation
- github.com/anthropics/anthropic-sdk-go v1.19.0 for Claude API
- github.com/spf13/cobra v1.10.2 for CLI framework
